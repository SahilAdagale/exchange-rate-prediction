
# 📈 Exchange Rate Prediction Using Machine Learning

## Overview

This project focuses on predicting currency exchange rates using Machine Learning techniques.
It analyzes historical exchange rate data and builds a predictive model to forecast future values.

The goal is to understand time-series behavior and apply ML models for real-world financial prediction.


## Features

* Uses historical currency exchange rate data
* Data preprocessing and cleaning
* Feature engineering
* Machine Learning model training
* Performance evaluation
* Exchange rate prediction

* Model performance is evaluated using:
